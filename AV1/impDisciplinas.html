Importar Disciplinas



O arquivo deverá estar no formato CSV com o seguinte layout:
id;nome_da_disciplina;periodo;idPreRequisito;creditos
id;nome_da_disciplina;periodo;idPreRequisito;creditos
id;nome_da_disciplina;periodo;idPreRequisito;creditos
O id deverá ser 0 para novos registros (cujos ids serão gerados automaticamente), caso esteja preenchido com algo diferente de 0, o registro correspondente terá seus valores substituídos pelos valores preenchidos.
Exemplo:
0;1FAC;1;0;10
0;1MAC;1;0;5
0;1ORG;1;0;10
2;1MAB;1;0;5
